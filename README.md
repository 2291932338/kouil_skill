# Hermes portable profile snapshot

This repository is maintained automatically from `/home/ubuntu/.hermes`.

Last sync: 2026-05-22T11:58:47.841020+00:00

## What is included

- `skills/` — active Hermes skills
- `plugins/` — user plugins when present
- `config/config.yaml` — sanitized Hermes config
- `config/.env.example` — environment variable names with values redacted
- `cron/` — cron metadata when present, with obvious secrets redacted
- `root-files/` — portable root-level persona/context files when present

## What is intentionally excluded

- `.env`, `auth.json`, OAuth tokens, API keys and passwords
- sessions, logs, caches, images/audio cache
- virtualenvs and generated bytecode

Use this as a migration snapshot. After cloning elsewhere, restore secrets manually into `.env`.
